Parrafo

Un parrafo consta de una o mas oraciones que forman una unidad autonoma de discurso.El inicio de un parrafo se indica con una nueva linea.

El texto es mas facil de entender cuando se divide en unidades de texto.Por ejemplo, un libro puede tener capitulos.Los capitulos pueden tener subtitulos.Bajo cada encabezado puede haber uno o mas parrafos.
